# ECOMMERCE-SALES-DASHBOARD
Interactive Sales Analytics Dashboard 
skills: Excel, Power BI, Dax Queries

This project presents an end-to-end E-Commerce Sales Analysis Dashboard designed to track, analyze, and visualize key business metrics such as revenue, profit, sales quantity, category performance, and state-wise insights.
The dashboard is fully interactive and designed for quick decision-making using filters, slicers, and drill-down capabilities.
